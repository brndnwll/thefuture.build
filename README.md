# Build the Future Website
Website for Build the Future: UC Berkeley’s student-run entrepreneurship hub.

## Development

### Dependencies

- Hugo v0.80.0

### Setup

Install all npm dependencies

```bash
yarn
```

Run development server 

```bash
yarn dev
```
