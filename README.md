Website for [Build the Future](https://thefuture.build): UC Berkeley’s student-run entrepreneurship hub.

## Development

### Dependencies

- Hugo v0.80.0

### Setup

Install all npm dependencies

```bash
yarn
```

Run development server

```bash
yarn dev
```
